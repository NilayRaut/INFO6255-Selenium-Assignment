 Description

The framework automates key student services across NEU platforms using:

    Java with Maven
    Selenium WebDriver
    TestNG as the test runner
    Excel-driven test inputs
    Automated screenshots
    HTML-style reporting

Test Scenarios

    Scenario 1 – Download latest transcript from Student Hub
    Scenario 2 – Add Canvas Calendar To-Do tasks
    Scenario 3 – Reserve study room at Snell Library
    Scenario 4 – Download dataset from NU Library DRS
    Scenario 5 – Update and verify Academic Calendar
